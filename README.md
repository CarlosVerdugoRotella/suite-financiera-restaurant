# Suite Financiera Restaurant

Herramientas integradas para planificación financiera de restaurantes.

## Archivos:
- `index.html` - Hub principal
- `calculadora-ingresos-m2.html` - Calculadora capacidad e ingresos
- `calculadora-inversion.html` - Modelo financiero inversión

## Uso:
1. Abrir `index.html` en navegador
2. Seguir flujo: Ingresos → Inversión
3. Los datos se transfieren vía localStorage

## Deploy:
GitHub Pages activado en Settings → Pages → main → /root
